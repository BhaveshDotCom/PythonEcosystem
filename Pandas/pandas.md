Pandas is an open-source Python library used for working with structured (tabular) data. It provides powerful tools to read, clean, manipulate, analyze, and store data efficiently.

https://pandas.pydata.org/about/


### Data Structures

***Series***
- A Series is a 1-dimensional labeled array, like a single column in Excel.

***DataFrame***
- A DataFrame is a 2-dimensional table with rows and columns.
